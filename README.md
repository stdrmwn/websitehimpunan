## Installation
**Git Clone**
```
git clone https://github.com/HarukiDev/AITalentDNA.git
```
**Go to the Directory**
```
cd frontend
```
**Install NPM**
```
npm i
```
**How to run**
```
npm run dev
```
**Endpoints**
[http://localhost:5173/](http://localhost:5173/)
